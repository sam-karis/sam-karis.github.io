About Hello Books

Hello Books is a web app that is created to bridge the gap of knowledge by making books available to everybody. In the world we live in today for anyone to survive alot and make an impact needs diverse knowledge and skills while resources are limitted. Hello Books bridge this gap by making reading resources available.

Services Offered By Hello Books
Search and View Books
All users can search and view the books available for our users.
Borrow a Book
Registered users can Borrow a book in the library.
Return a Book
A user who has borrowed a book can return the book using the app.
View Borrowing History
A registered user can view his/her borrowing history
